## How to Deploy

### Local Development (no changes needed)
./gradlew bootRun
cd react-ui-svc && npm start

### Production
export REACT_APP_API_BASE_URL=https://api.yourdomain.com
export CORS_ALLOWED_ORIGINS=https://yourdomain.com
./gradlew bootRun

### Docker
docker run -e CORS_ALLOWED_ORIGINS=https://yourdomain.com \
           -e PORT=8080 \
           myapp:latest

See ENVIRONMENT_CONFIGURATION.md for detailed deployment instructions.
